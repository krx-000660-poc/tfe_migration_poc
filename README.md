### Workspace rule

{서비스}{용도}\_{대분류}\_{중분류}\_{소분류}

- 서비스 : cjfw

- 용도

  | 용도                     | 표기  |
  | ----------------------- | ---- |
  | 공통 (ALL)               | A    |
  | 개발 (Development)       | D    |
  | 검증 (Quality Assurance) | Q    |
  | 관리 (Management)        | M    |
  | 운영 (Production)        | P    |
  
  |Environment |	Code|
|Development|	dev|
|Test	|test|
|System Integration Testing	|sit |
|User Acceptance Test	|uat|
|Staging |	stag |
|Pre-production	|pre |
|Production	|prod|

- 대분류 :

  - Private
  - Public
  - ALL
  - {Region code}

-  중분류 :

  - 용도 : `before` `after`
  - 서비스 단위 : `net` `inf` `db`

- 소분류 :  리소스 이름
